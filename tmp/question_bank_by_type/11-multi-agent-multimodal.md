# 多 Agent 与多模态

频次：

- `多 Agent 协作`：`12`
- `多模态`：`12`

多 Agent 核心问题：

1. 什么情况下值得上多 Agent？
2. 多 Agent 和单 Agent 的收益与成本是什么？
3. 多 Agent 状态共享怎么做？
4. 多 Agent 通信协议怎么设计？
5. 多个 Agent 结果冲突时谁来裁决？
6. 如何避免死循环、重复执行和互相推诿？

多模态核心问题：

1. 多模态 RAG 和纯文本 RAG 有什么不同？
2. 图片、截图、图表如何建索引？
3. CLIP 适合解决什么问题？
4. 给定软件截图，如何通过 RAG 帮用户理解界面组件？
5. 多模态用户信息如何存储和使用？

回答抓手：

- 多 Agent 先讲收益，再讲复杂度和裁决机制
- 多模态重点讲模态转换、索引表示、评测方式

# 编码题与手撕题  (面试题整理版)

## 1. 开发岗为什么还会问编码题？

因为编码题是最直接的基本功筛选器。很多 AI 开发岗并不会追求很难的算法题，但一定会看你写代码是否稳、复杂度是否清楚、边界条件是否完整。

## 2. 高频方向

1. `LRU`
2. 字符串处理
3. 树的序列化 / 反序列化
4. 并发任务调度
5. 数组、哈希、链表基础题

## 3. 面试官最常追问的点

1. 时间复杂度和空间复杂度是多少？
2. 边界条件怎么处理？
3. 如果上生产，这段代码还要补什么？
4. 并发版本和单线程版本差别是什么？

## 4. 你真正要准备的不是“刷很多题”，而是这几种能力

1. 读题快
2. 建模快
3. 写法稳
4. 边界全
5. 能把思路讲清楚

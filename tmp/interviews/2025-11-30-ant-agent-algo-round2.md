## 原始问题梳理

1. Agent 系统 Prompt 怎么设计和迭代，是否做过自动优化
2. 用户请求不完整时，如何补全用户意图
3. 构建 Agent 时遇到过哪些瓶颈
4. LangChain 的 memory 在多用户并发下如何做隔离，如何保证线程安全
5. 微调 Llama2 时如何选择和清洗训练样本

7. DPO 相比 SFT 的优劣，在 Agent 任务上效果是否明显
8. 偏好对如何构造，是自动还是人工
9. 为什么选择 vLLM，KV-cache 如何帮助推理加速
10. 自己做过哪些推理优化
11. 如果要支持 Streaming 输出但延迟超标，如何折中
12. 多轮对话状态如何管理，高并发下如何保证一致性
13. 外部工具调用链条如何做故障容错和超时机制
14. 工具调用失败后的 feedback 策略如何设计
15. 如何从用户行为日志抽取训练对话，是否做过归一化或事件抽象
16. Agent 哪些模块最容易在线上出问题，如何监控与定位

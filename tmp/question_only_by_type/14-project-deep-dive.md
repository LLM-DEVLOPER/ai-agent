# 项目深挖

## 1. 起手问题

1. 项目目标是什么？
2. 你真实负责什么？
3. 为什么这么选型？
4. 效果怎么量化？
5. 最难的 bad case 是什么？
6. 上线后出过什么事故？
7. 如果重做一遍你会改什么？

## 2. 顺着追的问题

1. 数据从哪里来，怎么清洗？
2. 为什么选这个模型和框架？
3. 为什么不用别的方案？
4. 你真正改动的模块是什么？
5. 你有没有失败案例？

## 3. 讲项目时必须覆盖的问题

1. 业务场景是什么？
2. 我的职责是什么？
3. 系统架构是什么？
4. 关键技术点是什么？
5. 指标和结果是什么？
6. 问题与复盘是什么？
